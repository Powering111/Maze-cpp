<h1> This is a simple maze game.</h1> 


	' ' is empty place, you can go there.
	'#' is wall.
	'S' is starting point.
	'E' is ending point.
	'P' shows where you are.

&nbsp;&nbsp;you can go up, left, down, right with <b>WASD</b>.
